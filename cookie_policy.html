Política de Cookies.

¿Qué es una cookie?

Una cookie es un archivo que un servidor web puede dejar en el dispositivo (ordenador, smartphone, tableta, etc.) que el usuario utiliza para conectarse, con el fin de identificar al usuario cuando se vuelva a conectar y/o para facilitar el uso y la navegación por el sitio web, proporcionando diferentes ventajas para los servicios interactivos.

Las cookies, por ejemplo, se utilizan para gestionar la sesión del usuario (por ejemplo, reduciendo el número de veces que debe introducir la contraseña) o para ajustar los contenidos de un sitio web a sus preferencias. Las cookies pueden ser de sesión, por lo que se eliminan una vez que el usuario abandona el sitio web que las generó, o persistentes, que permanecen en su dispositivo hasta una fecha determinada.

La mayoría de los navegadores aceptan el uso de cookies de forma automática, pero el usuario puede ajustar la configuración de su navegador para que la pantalla del ordenador le informe de la recepción de cookies y evitar que se instalen en su dispositivo. En este caso, la velocidad y la calidad de funcionamiento de los servicios de este sitio web pueden disminuir.

¿Qué tipo de cookies utiliza este sitio web?

Este sitio web utiliza diferentes tipos de cookies, con las finalidades que se describen a continuación:

Cookies técnicas: permiten al usuario navegar por la web y utilizar las diferentes opciones que ésta ofrece, como el inicio de sesión, la gestión de los parámetros del usuario o compartir contenidos a través de las redes sociales.

Cookies de análisis: pueden utilizarse para el seguimiento y análisis del comportamiento de los usuarios del sitio web. The information collected with this kind of cookie is used to measure website activity, in order to implement improvements based on analysis of data on the use the users make of it (for example, Google Analytics). These are persistent cookies.

Cómo controlar y eliminar las cookies

Si el usuario lo desea, puede desactivar y/o eliminar las cookies. Como información adicional para el usuario, le facilitamos dónde puede encontrar la configuración para gestionar las cookies en los principales navegadores.

As additional information for the user, we provide where they can find the settings to manage cookies on main browsers.

Google Chrome: Ajustes. Opciones avanzadas. Privacidad. Cookies.

Internet Explorer: Herramientas. Opciones de internet. Privacidad. Avanzadas.

Firefox: Opciones. Privacidad.

Safari: Preferencias. Privacidad.
